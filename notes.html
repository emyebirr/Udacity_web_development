Tree structures

Tree structures are a way of organizing data hierarchically. Each item in a tree is called a node, and each node can have zero or more child nodes.

  • The top node in a tree is called the root.
  • Nodes with no children are called leaves.
  • Each node has a parent, except for the root node.
